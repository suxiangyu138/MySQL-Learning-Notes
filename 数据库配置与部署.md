# Windows 版 MySQL 配置与部署 完整版（纯Windows、考试/实操通用）
## 一、Windows 常见部署方式
1. **安装版（MSI 安装包）**
   图形化下一步安装，自带服务、开机自启，新手常用。
2. **压缩包免安装版（ZIP）**
   绿色免装、可便携、自定义配置，教学/考试最常用。
3. **Docker 部署**
   适合本地开发，不用配置环境。

---

## 二、Windows 压缩包版 部署流程（必考+实操）
1. 解压 MySQL 压缩包到纯英文路径（无中文、无空格）
2. 在根目录新建配置文件：**`my.ini`**
3. 配置初始化、环境变量
4. 管理员CMD初始化、安装服务、启动
5. 修改密码、开启远程访问

---

## 三、核心配置文件 `my.ini`（Windows 专用）
放在MySQL根目录，直接复制可用：
```ini
[mysqld]
# 端口
port=3306
# 安装根目录
basedir=D:\mysql-8.0
# 数据存放目录
datadir=D:\mysql-8.0\data
# 字符集
character-set-server=utf8mb4
# 默认存储引擎
default-storage-engine=InnoDB
# 最大连接数
max_connections=200
# 关闭严格模式（可选）
sql_mode=NO_ENGINE_SUBSTITUTION

[mysql]
# 客户端默认字符集
default-character-set=utf8mb4
```

## 四、Windows CMD 关键操作（管理员身份运行）
### 1. 初始化（生成data目录+初始密码）
```cmd
mysqld --initialize --console
```
执行后会输出**临时初始密码**，复制保存。

### 2. 安装 Windows 系统服务
```cmd
mysqld --install mysql8
```

### 3. 启动 / 停止 服务
```cmd
net start mysql8
net stop mysql8
```

### 4. 登录 + 修改初始密码
```cmd
mysql -u root -p
```
登录后执行：
```sql
ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';
```

## 五、Windows 开启远程连接（局域网可用）
### 1. 授权 root 任意IP访问
```sql
USE mysql;
UPDATE user SET host = '%' WHERE user = 'root';
FLUSH PRIVILEGES;
```

### 2. Windows 防火墙放行 3306 端口
1. 控制面板 → 防火墙 → 高级设置
2. 新建**入站规则**：端口 3306，允许连接
3. 局域网其他电脑/Navicat 即可连接

---

## 六、Windows 环境变量配置
1. 此电脑 → 右键属性 → 高级系统设置 → 环境变量
2. 系统变量 `Path` 新增：
```
D:\mysql-8.0\bin
```
作用：任意文件夹CMD直接使用 `mysql`、`mysqld` 命令。

---

## 七、Windows 下 MySQL 日志说明
1. **错误日志**
   默认在 `data` 目录下 `.err` 文件，排查启动报错。
2. **慢查询日志 / 二进制日志**
   在 `my.ini` 中手动开启，开发环境默认关闭。

---

## 八、Windows 备份与恢复
### 1. 备份（cmd）
```cmd
mysqldump -u root -p 数据库名 > D:\backup\db.sql
```
### 2. 恢复
```cmd
mysql -u root -p 数据库名 < D:\backup\db.sql
```

---

## 九、常见问题（Windows专属）
1. **服务启动失败**
  - 路径有中文/空格
  - `my.ini` 路径配置错误
  - 未管理员运行CMD
2. **忘记密码**
   跳过权限启动 → 清空密码 → 恢复正常启动
3. **Navicat连不上**
  - 未改 `host=%`
  - 防火墙未放行3306
  - 服务未启动

---

## 十、背诵极简总结（考试用）
1. Windows 配置文件：**`my.ini`**
2. 核心配置：端口3306、`utf8mb4`、InnoDB引擎
3. 压缩包部署三步：**初始化 → 装服务 → 启服务**
4. 远程访问：修改 `user` 表 `host=%` + 放行防火墙端口
5. Windows 服务控制：`net start / net stop`
